# 후크나이츠 튜토리얼 설계안 (World 0 · 0-1 ~ 0-7)

> 원칙: 한 판 = **새 영웅 1명 획득 + 새 메커니즘 1개 학습**. 앞에서 배운 건 복습은 가볍게, 새 요소만 강제 코칭. 0-7 클리어 시 튜토리얼 완료 + 뽑기권 50.
> 코칭 방식: 설명이 뜨면 전체 정지(적·DP·SP), 지정 동작만 허용. 배치 후 방향 변경 불가는 0-1에서 1회 고지.

---

## 한눈에 보기

| 판 | 획득 영웅 | 직군/타입 | 새로 배우는 핵심 | 맵 | 신규 적 |
|----|----------|----------|------------------|-----|---------|
| 0-1 | 겁만 | 탱커 / AD | 선택·지상 배치·방향·확정·막기·스킬 | 세로 직선(현재) | 일반 |
| 0-2 | 리용세 | 서포터 / AP | **단상 전용 배치**, 서포터 힐, 탱+힐 조합 | 세로 직선 | (없음) |
| 0-3 | 동궈 | 뱅가드 / AD | **DP 경제**, 뱅가드 DP충전, **퇴각** | 세로 직선(저DP) | (없음) |
| 0-4 | 자고로 | 원거리 / AD | 원거리 부채꼴 딜, 방향 중요, 처치 우선 | 세로 직선 | 날쌘(빠름) |
| 0-5 | 보쌈 | 원거리 / AP | **데미지 타입(AD/AP·방어/마저)**, 조준 광역 스킬 | 세로 직선 | 묵직(방어↑) |
| 0-6 | 정글 | 근거리 / AD | 브루저, **끌어오기**, **코너 지형** | L자 코너(신규) | (혼합) |
| 0-7 | 후제 | 원거리 / AP | **보스전**, 전체 도트, 오토, 종합 운용 | 큰 맵/좌우 두 문(신규) | 보스 |

## 판별 상세

### 0-1 · 겁만 (탱커) — 이미 구현됨
- **배움**: 카드 선택 → 지상 배치 → 방향(↑) → 확정(같은 방향 재탭) → 막기 → 스킬. "배치 후 방향 변경 불가" 고지.
- **맵**: 세로 직선(스폰 위 1곳 → 넥서스 아래). 가운데 길(지상), 양옆 단상.
- **적**: 일반 슬라임 2.
- **코칭(6스텝)**: 환영 → 겁만 선택 → 빛나는 칸 배치 → 방향 ↑확정 → SP 충전 대기 → 스킬.

### 0-2 · 리용세 (서포터)
- **배움**: **원거리·서포터는 단상에만** / 탱커는 지상에만(반대 규칙 각인). 서포터 힐로 탱커 유지.
- **맵**: 세로 직선(단상 강조).
- **적**: 일반 3 — 겁만 혼자선 체력이 닳아 힐이 필요하게 설계.
- **코칭**: 겁만 지상 배치(빠른 복습) → "리용세는 단상! 지상은 안 돼요"(지상 한번 막고 단상 유도) → 리용세 단상 배치 → 힐 스킬.

### 0-3 · 동궈 (뱅가드)
- **배움**: **DP 경제** — 시작 DP를 낮춰 압박. 뱅가드는 저코스트 선봉이며 스킬로 **DP 충전**. **퇴각**으로 유닛 회수·재배치.
- **맵**: 세로 직선 (START_DP 낮춤).
- **적**: 일반 다수 빠르게 (DP 부족 체감).
- **코칭**: "DP가 부족해요" 인지 → 동궈 싸게 배치 → 동궈 스킬로 DP 회복 → 퇴각 버튼 시연.

### 0-4 · 자고로 (원거리 AD)
- **배움**: 원거리 딜러가 **단상에서 부채꼴 방향**으로 딜. 방향을 경로 쪽으로. 빠른 적은 막기보다 **처치**.
- **맵**: 세로 직선 (단상 위치로 경로 사격).
- **적**: 일반 + **날쌘**(빠름·leak2) — 탱커만으론 새므로 원거리 딜 필요.
- **코칭**: 자고로 단상 배치 → 방향을 경로 쪽으로 → "빠른 적은 사격으로 줄이세요".

### 0-5 · 보쌈 (원거리 AP)
- **배움**: **데미지 타입** — 물리(AD)는 방어, 마법(AP)은 마저에 막힘. **묵직**(방어 높음)은 AD가 안 박히고 AP가 강함. **조준 광역 스킬**(눈보라=타일 지정).
- **맵**: 세로 직선.
- **적**: **묵직**(방어↑·leak3) 위주 + 일반. 자고로(AD)로 안 죽는 묵직을 보쌈(AP)로 처리.
- **코칭**: "묵직은 방어가 높아 물리가 약해요" → 보쌈(AP) 단상 배치 → 눈보라를 적 몰린 칸에 조준 시전.

### 0-6 · 정글 (근거리)
- **배움**: 근거리 브루저(막으며 딜). **끌어오기**(죽음의 손아귀=적 1기 끌어와 묶기). **코너 지형**(길이 꺾임) 이해.
- **맵**: **신규 L자 코너** — 스폰 위 → 중간서 꺾여 → 넥서스. 코너 안쪽에 단상 포인트.
- **적**: 일반 + 날쌘(코너서 새려는 것).
- **코칭**: 정글 지상 배치 → 새는 날쌘을 끌어오기로 잡기 → 코너 위치 활용 안내.

### 0-7 · 후제 (원거리 AP) — 졸업전
- **배움**: **보스전**(보스 슬라임·leak5·체력 큼). **전체 도트**(부패의 씨앗). **오토 ON** 활용. 6명 종합 운용.
- **맵**: **신규 — 큰 맵 / 좌우 두 문**(좌·우에서 적 진입 → 넥서스). 멀티 레인 입문.
- **적**: 혼합 웨이브 + 보스 1.
- **코칭**: 배운 것 종합 배치(자유, 가벼운 안내) → 후제 부패의 씨앗 광역 도트 → 오토 ON → 보스 처치.
- **완료 보상**: 후제 + **뽑기권 50** + 튜토리얼 종료(버튼 → 테스트 전투로 전환).

## 제작 관점 메모 (구현 난이도)

이미 엔진에 있는 것(추가 개발 거의 없음):
- 단상/지상 직군 규칙, 방향 부채꼴, 막기(앞칸), 힐(리용세), DP충전(동궈), 퇴각, 조준 스킬(보쌈 타일/정글 적), 데미지 타입(AD/AP·방어/마저), 슬라임 4종(일반/날쌘/묵직/보스).

새로 필요한 것:
1. **스테이지별 데이터 분리**: `TUT_HEROES`(그 판에서 쓸 카드—신규+필요한 기존), `TUT_WAVES`, `START_DP`, 맵 파일을 판별로.
2. **신규 맵 2장**: 0-6 L자 코너, 0-7 큰 맵(좌우 두 문). — 0-1~0-5는 현재 세로 맵 재사용.
3. **다중 스폰(좌우 두 문)**: 0-7용 — 스폰 2곳에서 각각 최단경로 BFS, 적을 양쪽에서 생성. (현재는 단일 스폰)
4. 판별 코칭 스크립트(`TUTSCRIPT["0-2"]` …) 작성.

## 진행 제안
- **A안(추천)**: 0-2부터 한 판씩 만들고 그때그때 확인받기 (코칭 톤·난이도 조정 쉬움).
- **B안**: 0-2~0-5(현재 맵 재사용분)를 먼저 묶어서 만들고 → 0-6,0-7(신규 맵) 별도.
